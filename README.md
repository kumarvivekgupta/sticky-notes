# sticky-notes
Sticky Notes has various important and exciting features.

TechStack: React, TailwindCss

Features:
1- Add a note
2- Edit a note
3- Pin a note
4- Check completed items
5- Sorting on pinned and created At.
6- Cool Animations

Major plugins:
- "tailwindcss": "^3.1.2",
-  "nanoid": "^4.0.0",
-   "bootstrap-icons": "^1.8.3",

Page Load Time (Lighthouse)
1- FCP: 1.6s
2- LCP: 1.9s
3- TBT: 0.34s
